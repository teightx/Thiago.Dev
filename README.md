## 🔧 Instalação e Uso

```bash
# Clone o repositório
git clone https://github.com/teightx/Thiago.Dev.git

# Entre na pasta
cd Thiago.Dev

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```
